<img src="/static/images/icons/job.png" /> List all updated files with the following data:
        
* **State**: Environment. 
       
* **Project**: Project name.
  
* **Item**: File name.

* **Viewpath**: File path.

* **User**: User name.

* **Job**: Job name.

* **Job date**: Date at footprint operation.
